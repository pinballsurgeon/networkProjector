# Universal Network Mirror v2.0 Backup (2025-11-24)

This is a snapshot of the Universal Network Mirror extension development.

## Features Included
1. **Solar System Physics:** Root Domains (Planets) and Subdomains/Junk (Moons).
2. **Directional Traffic:** Outgoing (Sun->Planet) and Incoming (Space->Planet).
3. **Decay Engine:** Objects dissolve over time (10s window).
4. **God Mode HUD:** Interactive inspection with grading (A-F) and bloat analysis.
5. **Deployment Fixes:** Simplified Service Worker architecture (no modules).

## Installation
Load this folder as an Unpacked Extension in Chrome.
